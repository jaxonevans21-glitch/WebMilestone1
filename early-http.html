Early HTTP

HTTP/0.9

HTTP/0.9 was the first version and was very basic. It could only handle simple GET requests, which means it could only load plain text from a server.

  • Only supported GET requests
  • No images or media
  • Very limited features

HTTP/1.0

HTTP/1.0 improved a lot compared to 0.9. It added more structure and allowed websites to be more interactive.

  • Added status codes (like 404 errors)
  • Introduced headers
  • Supported images and other content types
